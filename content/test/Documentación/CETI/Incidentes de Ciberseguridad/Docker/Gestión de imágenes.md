```
docker image
```

Listar imágenes:

```
docker images
```

Borrar imagen:

```
docker rmi <imagen>
```

Descargar imagen:

```
docker pull <imagen>
```

Borrar todas las imágenes que no estén siendo usadas:

```
docker image prune -a
```
